# Bluestock Mutual Fund Analytics Capstone Project

## Project Overview

This project focuses on analyzing mutual fund performance, investor behavior, portfolio risk, and industry trends using Python and Power BI. The solution integrates ETL processes, exploratory data analysis, financial performance metrics, advanced risk analytics, and interactive dashboards.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook
* Power BI
* Git & GitHub

## Project Modules

* Data Cleaning & ETL
* Exploratory Data Analysis
* Performance Analytics
* Risk Metrics
* Investor Analytics
* Dashboard Development

## How to Run

1. Install required Python libraries.
2. Execute scripts using `run_pipeline.py`.
3. Open generated reports and dashboards.
4. Review Power BI dashboard for visual insights.
